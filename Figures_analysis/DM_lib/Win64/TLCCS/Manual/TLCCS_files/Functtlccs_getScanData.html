tlccs_getScanData

ViStatus tlccs_getScanData (ViSession instrumentHandle, ViReal64 _VI_FAR data[]);

Purpose

This function reads out the processed scan data.
Note:
When the raw scan data is overexposed, so that a proper data processing is not possible, the function returns
VI_ERROR_SCAN_DATA_INVALID
and all data points are set to zero (0.0).

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
Output
Name Type Description
data ViReal64[] This parameter returns the processed scan data.
Note:
The array must contain at least CCS_SERIES_NUM_PIXELS (3648) elements (ViReal64[CCS_SERIES_NUM_PIXELS]).

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
