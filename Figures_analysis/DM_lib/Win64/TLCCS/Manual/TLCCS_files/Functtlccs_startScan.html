tlccs_startScan

ViStatus tlccs_startScan (ViSession instrumentHandle);

Purpose

This function triggers the the CCS to take one single scan.
Note:
The scan data can be read out with the function 'Get Scan Data'
Use 'Get Device Status' to check the scan status.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.

Return Value

Name Type Description
status ViStatus This is the error code returned by the function call. For error codes and descriptions see <Error Message>.
