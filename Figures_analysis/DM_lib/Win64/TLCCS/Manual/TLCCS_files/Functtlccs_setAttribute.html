tlccs_setAttribute

ViStatus tlccs_setAttribute (ViSession instrumentHandle, ViAttr attribute, ViAttrState value);

Purpose

Sets a specified attribute for the given instrument session.

Parameters

Input
Name Type Description
instrumentHandle ViSession This parameter accepts the instrument handle returned by <Initialize> to select the desired instrument driver session.
attribute ViAttr Specifies the attribute whose value you want to set.
Attributes:
CCS_SERIES_ATTR_USER_DATA
Description:  This attribute value is the data used privately by the application for a particular session. This data is not used by the instrument driver for any purposes. It is provided to the application for its own use.
value ViAttrState The value to which to set the specified attribute.

Return Value

Name Type Description
status ViStatus This value shows the status code returned by the function call.
For Status Codes see function <Error Message>.
